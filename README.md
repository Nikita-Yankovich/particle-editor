# particle-editor



### Описание:



particle editor (редактор частиц) - программа, созданная для отрисовки эффектов и работы с ними.



Частицы - это очень гибкий инструмент в разработке игр, использование которого позволяет сделать сложные и зрелищные анимации, не задействуя больших вычислительных мощностей.



Примеры использования частиц и подробное описание продукта прилагаются в технической документации documentation.pdf (сайт? - на будущее)



### Запуск программы:



Покуда программа находится на этапе разработки, поэтому запуск возможен только через установленный компилятор Python.



Проверка работоспособности программы осуществляется путём запуска main.py в корневой папке проекта.

Успешное выполнение этого файла отмечает отсутствие конфликтов взаимосвязей в бизнес-логике программы.



Чтобы протестировать работу конкретных примеров использования редактора частиц, можно воспользоваться готовыми файлами с тестовыми анимациями из каталога test\&examples/main# простым запуском через Python.



Также, программа даёт возможность написать свою анимацию с частицами, взяв за основу одну из имеющихся программ-примеров или воспользовавшись документацией.



Исходный код продукта можно просмотреть через Python IDLE или любой другой текстовый редактор, среду разработки.
